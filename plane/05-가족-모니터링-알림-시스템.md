# 가족 모니터링 및 알림 시스템 기획서

> **"떨어져 있어도 부모님을 안심하고 지켜볼 수 있도록"**

---

## 📋 개요

### 배경

**가족(자녀)의 고민:**
- "부모님이 오늘 뭐 하셨는지 궁금해"
- "약은 제때 드셨을까?"
- "병원 예약은 어떻게 됐지?"
- "갑자기 쓰러지시면 어떡하지?"
- "매일 전화하면 잔소리 같고, 안 하면 불안해"

**시니어의 입장:**
- "자식들한테 매번 보고하기 싫어"
- "감시당하는 것 같아서 불편해"
- "내 프라이버시도 중요해"

### 우리의 해결책

**비침습적 모니터링:**
- 시니어가 자연스럽게 앱 사용 → 자동으로 가족에게 알림
- CCTV처럼 감시하는 게 아님
- 중요한 순간만 알림 (과도한 알림 X)

**3단계 알림 체계:**
1. **일상 알림** (매일): "아버지 이발 완료"
2. **주간 리포트** (주 1회): "이번 주 활동 요약"
3. **긴급 알림** (즉시): "12시간 앱 사용 없음"

## 🔔 알림 시스템 설계

### 알림 분류

#### 1. 일상 알림 (Low Priority)
**특징:**
- 소리 없음 (무음)
- 배지 숫자만 표시
- 모아서 확인 가능

**예시:**
- "아버지 아침 약 복용 완료"
- "이발 완료"
- "앱 사용 (30분 전)"

---

#### 2. 중요 알림 (Medium Priority)
**특징:**
- 소리 있음 (일반 알림음)
- 푸시 알림
- 즉시 확인 권장

**예시:**
- "아버지 약 복용 안 함 (30분 지남)"
- "예약 시간 놓침"
- "병원 예약 내일"

---

#### 3. 긴급 알림 (High Priority)
**특징:**
- 큰 소리 (긴급 알림음)
- 전체 화면 알림
- 즉시 확인 필수

**예시:**
- "12시간 앱 사용 없음"
- "긴급 SOS 버튼 눌림"
- "낙상 감지"

---

```python
# 알림 규칙

    }

```

## 🚨 긴급 상황 대응

### 긴급 상황 감지

#### 1. 장시간 미사용
```
12시간 앱 미사용
  → 시니어에게 자동 전화
  → 응답 없으면 가족 긴급 알림
```

#### 2. SOS 버튼
```
시니어 앱에서 SOS 버튼 누름
  → 즉시 가족 + 119 알림
  → GPS 위치 전송
```

#### 3. 낙상 감지 (향후 기능)
```
스마트워치 연동
  → 낙상 감지 AI
  → 1분 내 응답 없으면 자동 신고
```

## 🚀 로드맵

### Phase 1: 런칭 (1개월차)
- 기본 활동 내역 공유
- 일상/중요 알림
- 1:1 가족 연동

### Phase 2: 강화 (3개월차)
- 긴급 알림 시스템
- 주간 리포트
- 다중 가족 지원

### Phase 3: 고도화 (6개월차)
- AI 기반 이상 패턴 감지
- 스마트워치 연동 (낙상 감지)
- 예측 알림 ("내일 병원 예약 있어요")
