## I. 플랫폼 개요

### 1.1 플랫폼 컨셉

```
VIP 시니어 통합 플랫폼
= 바야다 전문 홈헬스케어 (핵심)
  + 일상생활 통합 서비스 (확장)
```

**핵심 가치**: 공감(Compassion), 탁월함(Excellence), 신뢰(Reliability)

### 1.2 서비스 구조

```
┌─────────────────────────────────────────────────────┐
│                  VIP 시니어 통합 플랫폼              │
└─────────────────────────────────────────────────────┘
                        │
        ┌───────────────┴───────────────┐
        │                               │
   [핵심 서비스]                   [확장 서비스]
        │                               │
        ↓                               ↓
┌──────────────────┐          ┌──────────────────┐
│ 바야다 전문      │          │ 일상생활 통합    │
│ 홈헬스케어       │          │ 서비스           │
├──────────────────┤          ├──────────────────┤
│                  │          │                  │
│ • 방문 간호      │          │ • 헬스케어       │
│   (케어 메이트)  │          │   (자가 관리)    │
│                  │          │                  │
│ • 방문 요양      │          │ • 식생활         │
│   (케어 기버)    │          │   (도시락/밑반찬)│
│                  │          │                  │
│ • AI 재활 운동   │          │ • 생활환경       │
│   (에버엑스)     │          │   (청소/수리)    │
│                  │          │                  │
│ • 원격 진료      │          │ • 이동지원       │
│   (K-DOC)        │          │   (택시/동행)    │
│                  │          │                  │
│ • 의료용품 배송  │          │ • 가족 모니터링  │
│   (DKSH)         │          │                  │
└──────────────────┘          └──────────────────┘
```

### 1.3 타겟 고객

#### 주 타겟: 중증 질환/장애 환자
- 뇌졸중, 심혈관 질환 등 중증 질환자
- 장애 진단을 받은 성인 및 노인
- 수술 후 재활이 필요한 환자
- 만성 질환으로 지속적 관리가 필요한 노인
→ **바야다 전문 홈헬스케어 제공**

#### 부 타겟: 일반 시니어
- 건강한 고령자
- 독립 생활 중인 시니어
- 경증 만성 질환자
→ **헬스케어 자가 관리 + 생활 서비스 제공**

#### 서브 타겟: 가족
- 원거리 거주 자녀
- 돌봄 부담이 큰 가족
→ **통합 모니터링 + 대리 예약/관리 제공**

### 1.4 비즈니스 모델

```
수익 구조 = 핵심 (안정적 수익) + 확장 (수수료 수익)

핵심 수익:
- 방문 간호/요양: 장기요양보험 기반
  → 월 1,400,000원/1인 (안정적)

확장 수익:
- 도시락/밑반찬: 플랫폼 수수료 10%
- 청소/수리: 플랫폼 수수료 15%
- 이동지원: 플랫폼 수수료 10%
- AI 재활 프리미엄: 월 50,000원
```

## II. 핵심: 바야다 전문 홈헬스케어

> **타겟**: 중증 질환/장애 환자
> **차별화**: 간호사(RN) 중심 전문 케어
> **수익**: 장기요양보험 기반 (안정적)

### 2.1 방문 간호 서비스 (케어 메이트)

#### 서비스 개요
**제공 인력**: 정규 간호대학 졸업 + 종합병원 경력 간호사(RN)
**법적 근거**: 방문간호지시서 (의사/한의사/치과의사 발급)
**제공 내용**: 간호처치, 재활지원, 질병예방, 생활습관 교육

#### 화면 플로우
```typescript
경로: /care/nursing

[방문간호지시서 업로드]
  ↓
[건강 상태 입력]
  - 주 진단명
  - 현재 증상
  - 필요한 간호처치
  ↓
[간호사 매칭]
  - AI 자동 추천
  - 전문 분야 매칭
  ↓
[일정 조율]
  ↓
[방문 시작]
  ↓
[방문 기록 작성] (간호사)
  ↓
[가족 앱 자동 전송]
```

*(상세 UI는 기존 VIP-홈헬스케어-통합-플랫폼-마스터-기획서.md 참조)*

### 2.2 방문 요양 서비스 (케어 기버)

#### 서비스 개요
**제공 인력**: 요양보호사
**차별화**: 초기 방문 시 **간호사 + 요양보호사 함께 방문**
**제공 내용**: 신체활동 지원, 일상생활 지원, 인지활동 지원, 정서 지원

#### 화면 플로우
```typescript
경로: /care/caregiving

[장기요양 인정 신청 대행] (무료)
  ↓
[등급 판정 대기]
  ↓
[초기 방문] (간호사 + 요양보호사)
  - 건강 상태 전문 평가
  - 일상생활 능력 평가
  - 주거 환경 확인
  ↓
[맞춤 케어플랜 수립] (간호사)
  ↓
[정기 방문 시작] (요양보호사)
  ↓
[방문 기록 작성]
  ↓
[월 1회 간호사 정기 점검]
```

*(상세 UI는 기존 VIP-홈헬스케어-통합-플랫폼-마스터-기획서.md 참조)*

### 2.3 AI 기반 가정 재활 (에버엑스 파트너십)

#### 서비스 개요
**파트너**: 에버엑스(EverEx) - 근골격계 디지털 헬스케어
**기능**: AI 맞춤형 재활 운동 + 실시간 자세 피드백
**콘텐츠**: 바야다 미국 본사 + 에버엑스 + 국내 재활 전문의 감수

#### 화면 플로우 (Phase 2)
```typescript
경로: /rehab/exercise

[AI 재활 프로그램 생성]
  - 환자 질환/나이/증상 분석
  - 맞춤 운동 구성
  ↓
[운동 영상 제공]
  - 3D 애니메이션 시연
  - 음성 가이드
  ↓
[실시간 자세 피드백]
  - 카메라 AI 분석
  - "팔을 조금 더 올려보세요"
  ↓
[운동 기록 자동 저장]
  ↓
[간호사 원격 모니터링]
  - 실천율 확인
  - 정확도 분석
  - 프로그램 조정
```

### 2.4 원격 진료 연동 (K-DOC 파트너십) - Phase 2

#### 서비스 개요
**파트너**: 케이닥(K-DOC) - 디지털 헬스케어
**기능**: 방문 간호 중 의사 실시간 연결
**통합 EMR**: 간호 + 요양 + 재활 + 진료 기록 통합

### 2.5 의료용품 배송 (DKSH 파트너십) - Phase 2

#### 서비스 개요
**파트너**: DKSH코리아헬스케어 - 물류/유통
**기능**: 간호 중 사용한 용품 자동 주문 + 다음 방문 전 배송

## III. 확장 1: 헬스케어 자가 관리

### 3.2 복약 관리 시스템

#### Phase 1: 즉시 개발 가능 ✅
- 약 수동 등록 (이름, 용량, 복용시간)
- 복용 시간 푸시 알림
- 복용 완료 체크
- 약 재고 자동 계산

#### Phase 2: AI 기능 추가 ⏸️

- 약물 상호작용 체크

### 3.3 건강 기록 관리

#### Phase 1: 수동 입력 ✅
- 혈압, 혈당, 체중, 체온 입력

- 그래프/차트 시각화
- 가족 공유

#### Phase 2: 자동 연동 ⏸️
- 스마트워치 자동 연동

### 3.4 응급 상황 대응

#### Phase 1: ✅
- SOS 버튼 (3초 길게 누르기)
- 가족 자동 알림 (푸시/SMS)
- 실시간 위치 공유

#### Phase 2: ⏸️
- 119 자동 다이얼

## IV. 확장 2: 식생활 통합 서비스

### 4.1 식단 기록 & 영양 분석

#### Phase 1: 즉시 개발 가능 ✅

```typescript
  ↓

  ↓

  ↓
[영양 분석 대시보드]
  - 오늘/주간/월간
  - 부족한 영양소 알림
  ↓
[GPT 기반 식단 추천]
```

**기능**:
- 식사 메뉴 수동 입력
- 영양 성분 자동 계산

#### Phase 2: AI 사진 인식 ⏸️

## V. 확장 3: 생활환경 관리

> **타겟**: 모든 시니어
> **목적**: 안전하고 쾌적한 주거 환경
> **특징**: 정보 제공(Phase 1) → 실제 서비스(Phase 2)

### 5.1 청소 서비스

#### Phase 1: 업체 정보 제공 ✅
```typescript
경로: /home/cleaning/info

[청소 업체 리스트]
  - 시니어 전문 업체
  - 평점/리뷰
  - 가격 정보
```

## XII. 수익 모델

### 12.1 핵심 수익 (안정적)

#### 방문 간호/요양 (장기요양보험)
```
1인당 월 수익:
- 방문 요양 (주 5회 × 2시간): 800,000원/월
- 방문 간호 (주 3회 × 1시간): 600,000원/월
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
통합 서비스: 1,400,000원/월/1인

1,000명 고객 기준:
14억원/월 = 168억원/년
```

### 12.2 확장 수익 (플랫폼 수수료)

#### 도시락/밑반찬 (Phase 2)
```
플랫폼 수수료: 주문 금액의 10%

예시:
- 도시락 1인 주문: 8,000원 × 30일 = 240,000원/월
- 플랫폼 수수료: 24,000원/월

1,000명 중 30% 이용 가정:
24,000원 × 300명 = 720만원/월
```

#### 청소/수리 서비스 (Phase 2)
```
플랫폼 수수료: 서비스 금액의 15%

예시:
- 청소 1회: 40,000원 × 15% = 6,000원
- 월 평균 2회 이용 = 12,000원/월/1인

1,000명 중 20% 이용 가정:
12,000원 × 200명 = 240만원/월
```

#### 이동지원 (Phase 2)
```
플랫폼 수수료: 서비스 금액의 10%

예시:
- 병원 동행 1회: 30,000원 × 10% = 3,000원
- 월 평균 2회 이용 = 6,000원/월/1인

1,000명 중 30% 이용 가정:
6,000원 × 300명 = 180만원/월
```

#### AI 재활 프리미엄 (Phase 2)
```
프리미엄 플랜: 50,000원/월
(1:1 PT 영상 통화, 맞춤 프로그램)

방문 간호 이용자 중 20% 전환 가정:
50,000원 × 200명 = 1,000만원/월
```

### 12.3 총 예상 매출 (1,000명 고객 기준)

```
Phase 1 (방문 간호/요양만):
14억원/월 = 168억원/년

Phase 2 (확장 서비스 추가):
14억원 (핵심)
+ 720만원 (도시락)
+ 240만원 (청소)
+ 180만원 (이동)
+ 1,000만원 (AI 재활)
━━━━━━━━━━━━━━━━━
= 15.14억원/월 = 182억원/년
```

## XIII. 경쟁 우위

### 13.1 vs 일반 요양 업체

| 구분 | 일반 요양 업체 | VIP 통합 플랫폼 |
|-----|---------------|----------------|
| **초기 방문** | 요양보호사만 | 간호사 + 요양보호사 |
| **케어플랜** | 표준 플랜 | 간호사 맞춤 설계 |
| **모니터링** | 전화 연락 | 앱 실시간 확인 |
| **재활** | 없음 또는 별도 | AI 재활 포함 |
| **일상생활** | 요양만 | 식·주·이동 통합 |
| **가족 소통** | 전화 | 앱 + 채팅 + 알림 |

### 13.2 vs 일반 시니어 앱

| 구분 | 일반 시니어 앱 | VIP 통합 플랫폼 |
|-----|---------------|----------------|
| **타겟** | 건강한 시니어 | 중증 환자 + 일반 시니어 |
| **핵심 서비스** | 정보 제공 | 전문 홈헬스케어 |
| **수익 모델** | 광고/수수료 | 장기요양보험 (안정적) |
| **파트너십** | 일반 업체 | 에버엑스/K-DOC/DKSH |
| **차별화** | UX | 간호사 중심 전문 케어 |
