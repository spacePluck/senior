2025년 1월 24일

## 🎯 제안 배경

### 커먼키친의 강점
```
✓ 건강한 도시락 전문 브랜드
✓ 영양사 감수 메뉴
✓ 신선한 재료, 당일 조리
✓ 전국 배송 네트워크
```

### 시니어 식사 시장의 기회
```
→ 1인 가구 시니어: 150만 명
→ 식사 준비 어려움
→ 영양 불균형 문제
→ 건강 식단 필요성
```

### 현재의 과제
```
✗ 기존 도시락은 시니어에게 과도한 양
✗ 나트륨/당분이 높음
✗ 씹기/삼키기 어려운 재료
✗ 만성질환 고려하지 않은 메뉴
```

### VIP Care의 솔루션
```
→ 시니어 맞춤 메뉴 개발
→ 건강 데이터 기반 추천
→ 원클릭 주문 시스템
→ 식사 기록 자동 저장
```

## 💡 제안 내용

### 1. 시니어 전용 메뉴 개발

#### "VIP 건강 밥상" 라인업
```
메뉴 특징:
  - 적정량 (300-400kcal/식)
  - 저염 (나트륨 800mg 이하)
  - 부드러운 조리 (씹기 쉬움)
  - 고단백 (근육 손실 예방)
  - 다양한 채소 (식이섬유)

메뉴 종류:
  1. 혈압 관리 메뉴
     - 저염, 칼륨 풍부
     - 예: 연두부 닭가슴살 덮밥

  2. 당뇨 관리 메뉴
     - 저당, 복합 탄수화물
     - 예: 현미밥 고등어 구이

  3. 소화 편한 메뉴
     - 부드러운 재료
     - 예: 야채 죽, 달걀찜

  4. 일반 건강 메뉴
     - 균형 잡힌 영양
     - 예: 연어 샐러드 볼
```

#### 영양 정보 상세 표시
```
각 메뉴마다 표시:
  - 칼로리
  - 나트륨
  - 당류
  - 단백질
  - 식이섬유
  - 주요 알레르기 성분
```

### 2. VIP Care 앱 통합

#### 주문 프로세스
```typescript
주문 흐름:

1. 개인 맞춤 추천
   - VIP Care 건강 데이터 기반
   - 혈압/혈당 고려
   - 복용 약물 고려
   - AI가 메뉴 추천

2. 원클릭 주문
   - 큰 버튼 UI
   - "오늘의 추천 메뉴 주문" 버튼
   - 단골 메뉴 저장

3. 자동 결제
   - VIP Care 등록 카드 사용
   - 또는 가족 카드로 결제

4. 배송 알림
   - 조리 시작 알림
   - 배송 출발 알림
   - 도착 30분 전 알림

5. 식사 기록
   - 식사 완료 체크
   - 맛/포만감 평가
   - 건강 기록에 자동 저장
```


```
  - VIP Care 사용자 10% 할인 → 72,000원

  - 월-금 점심+저녁 배송

주 7일 3식 플랜:
```
