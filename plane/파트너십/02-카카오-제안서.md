## 🎯 제안 배경

### 카카오모빌리티의 강점
```
✓ 국내 택시 호출 시장 점유율 90%
✓ 방대한 기사 네트워크
✓ 안정적인 기술 인프라
✓ 브랜드 신뢰도
```

### 시니어 시장의 기회
```
→ 65세 이상 인구: 1,000만 명
→ 택시 이용률: 주 2-3회 (고빈도)
→ 병원, 마트 등 정기적 이동 필요
→ 가족이 대신 호출/결제하는 니즈
```

### 현재의 과제
```
✗ 시니어에게 복잡한 앱 UI
✗ 가족 연동 기능 부족
✗ 안심 기능 부재 (위치 공유 등)
✗ 정기 이동 패턴 학습 X
```

### VIP Care의 솔루션
```
→ 시니어 친화 인터페이스
→ 가족 대리 호출 기능
→ 실시간 위치 공유
→ 단골 목적지 원클릭 호출
```

## 💡 제안 내용

### 1. API 직접 연동

#### Kakao Mobility API 활용
```typescript
통합 기능:

1. 택시 호출
   - VIP Care 앱 내에서 직접 호출
   - 카카오택시 앱 설치 불필요

2. 실시간 배차 정보
   - 택시 위치 추적
   - 도착 예정 시간

3. 자동 결제
   - VIP Care 내 카드 등록
   - 가족 카드로 결제 가능

4. 탑승 기록
   - 모든 이동 기록 저장
   - 가족과 자동 공유
```

#### 시니어 특화 기능
```
- 큰 글씨, 큰 버튼
- 단골 목적지 저장 (병원, 마트 등)
- 원클릭 호출
- 음성 명령 지원
```

### 2. 가족 대리 호출

#### 작동 방식
```
1. 자녀가 VIP Care 앱에서 부모님 택시 호출
2. 출발지: 부모님 집
3. 도착지: 병원 (미리 저장된 위치)
4. 결제: 자녀 카드
5. 알림: 부모님에게 "택시 곧 도착" 알림
6. 추적: 자녀가 실시간 위치 확인
```

#### 가족 안심 기능
```
- 실시간 위치 공유
- 탑승/하차 자동 알림
- 예상 도착 시간 알림
- 경로 이탈 시 알림
```

### 3. 시니어 전용 서비스

#### "카카오 실버 택시" (네이밍 협의 가능)
```
특징:
  - 시니어 응대 교육 이수 기사
  - 천천히, 안전 운전
  - 짐 들어주기 서비스
  - 문까지 에스코트

요금:
  - 일반 택시 + 소정의 추가 요금
  - 또는 동일 (서비스 차별화)
```
