## 📋 제안 개요

### 제안 기업
**VIP Care** (시니어 케어 통합 플랫폼)

### 제안 대상
**K-DOC** (대한민국 대표 원격 진료 플랫폼)

### 제안 일자
2025년 1월 24일

### 제안 목적

## 🎯 제안 배경

### K-DOC의 강점
```
✓ 국내 최대 원격 진료 플랫폼
✓ 20,000명+ 협력 의사 네트워크
✓ 비대면 진료법 시행 (2024년~)
✓ 검증된 의료 시스템
```

### 시니어 원격 진료 시장의 기회
```
→ 거동 불편 시니어: 200만 명
→ 병원 방문 어려움 (교통, 대기시간 등)
→ 만성질환 정기 진료 필요
→ 가족 대리 예약 니즈
```

### 현재의 과제
```
✗ 시니어에게 복잡한 앱 사용
✗ 건강 데이터 연동 부족
✗ 가족 참여 기능 미흡
✗ 복약 관리 연계 부족
```

### VIP Care의 솔루션
```
→ 시니어 친화 인터페이스
→ 자동 건강 데이터 전송
→ 가족 동시 참여 기능
→ 진료 후 자동 복약 등록
```

## 💡 제안 내용

### 1. 통합 원격 진료

#### VIP Care 앱 내 K-DOC 서비스 통합
```typescript
통합 프로세스:

1. 진료 예약
   - VIP Care 앱에서 직접 예약
   - K-DOC 앱 설치 불필요

2. 건강 데이터 자동 전송
   - 혈압, 혈당, 체중 등
   - 복약 기록
   - 증상 일지

3. 가족 동시 참여
   - 자녀가 진료에 함께 입장
   - 화면 공유 및 음성 참여

4. 진료 후 자동 처리
   - 처방전 자동 등록
   - 복약 알림 자동 설정
   - 진료비 자동 결제

5. 약 배송 연계
   - 처방약 자동 주문
   - 집으로 배송
```

#### 시니어 특화 기능
```
- 큰 화면, 큰 글씨
- 단순화된 예약 프로세스
- 음성 명령 지원
- 진료 전 체크리스트 안내
```

### 2. 건강 데이터 연동


### 3. 가족 참여 진료

#### "가족 동반 진료" 기능
```
작동 방식:

1. 시니어가 진료 예약
2. 가족 초대 링크 자동 발송
3. 진료 시간에 가족도 앱으로 입장
4. 의사-환자-가족 3자 화상 진료
5. 진료 내용 자동 녹화 (동의 시)
6. 가족에게 진료 요약 전송
```

#### 가족 안심 기능
```
- 진료 일정 자동 알림
- 진료 내용 요약 공유
- 처방 내역 실시간 확인
- 다음 진료 일정 알림
```
