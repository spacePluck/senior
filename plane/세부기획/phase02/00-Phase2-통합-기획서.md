## Phase 2 개요

### Phase 1 → Phase 2 전환

```
Phase 1 (정보 제공)              Phase 2 (실제 서비스)
─────────────────────────────────────────────────────

도시락 업체 정보        →        실제 도시락 주문
청소 업체 정보          →        실제 청소 예약
일반 택시 연동          →        시니어 전용 택시
                        →        방문 간호/요양 (신규)
```

### Phase 2 핵심 전략

#### 1. BD 완료 후 순차 오픈 🤝
- 도시락/밑반찬 업체 계약
- 청소/수리 업체 계약
- 시니어 택시 기사 모집
- 방문 간호사/요양보호사 채용

#### 2. 플랫폼 수수료 모델 💰
- 도시락/밑반찬: 10%
- 청소/수리: 15%
- 시니어 택시: 10%
- 병원 동행: 20%

#### 3. 안정적 수익 (바야다) 📈
- 방문 간호: 600,000원/월/1인
- 방문 요양: 800,000원/월/1인
- 장기요양보험 기반 (안정적)

#### 4. 파트너 앱 개발 👨‍⚕️
- 간호사/요양보호사용
- 방문 기록 작성
- 케어플랜 수립

### 3. 방문 요양 서비스 (케어 기버) ⭐⭐⭐

#### 서비스 개요
**제공 인력**: 요양보호사
**차별화**: 초기 방문 시 **간호사 + 요양보호사 함께 방문**
**제공 내용**: 신체활동 지원, 일상생활 지원, 인지활동 지원, 정서 지원

#### 개발 범위
```typescript
경로: /care/caregiving

[장기요양 인정서 확인]
  - 등급 확인 (1-5등급)
  - 이미 신청 대행한 경우 자동 연결
  ↓
[건강 상태 입력]
  - 일상생활 능력 체크리스트
  - 인지 기능 체크리스트
  ↓
[초기 방문 일정 잡기]
  ┌─────────────────────────────────────┐
  │  초기 방문 일정                      │
  │                                      │
  │  🌟 특별한 초기 방문                │
  │  간호사 + 요양보호사가 함께 방문    │
  │                                      │
  │  목적:                               │
  │  • 건강 상태 전문 평가 (간호사)     │
  │  • 일상생활 능력 평가 (요양보호사)  │
  │  • 주거 환경 확인                   │
  │  • 맞춤 케어플랜 수립 (간호사)      │
  │                                      │
  │  희망 일시: [날짜 선택]             │
  │                                      │
  │  [일정 잡기]                        │
  └─────────────────────────────────────┘
  ↓
[초기 방문 진행]
  - 간호사: 건강 평가 + 케어플랜 수립
  - 요양보호사: 일상생활 능력 평가
  ↓
[케어플랜 수립] (간호사)
  - 단기/장기 목표
  - 서비스 계획 (방문 횟수, 시간)
  - 위험도 평가 (낙상, 욕창 등)
  ↓
[정기 방문 시작] (요양보호사)
  - 매주 월~금 오전 10시
  - 방문 시간: 2시간
  ↓
[월 1회 간호사 정기 점검]
  - 건강 상태 재평가
  - 케어플랜 조정
```

#### 핵심 기능
- ✅ 장기요양 인정서 연동
- ✅ 일상생활/인지 기능 평가
- ✅ 초기 방문 예약 (간호사 + 요양보호사)
- ✅ 케어플랜 수립 시스템 (후술)
- ✅ 정기 방문 일정 관리
- ✅ 월 1회 간호사 점검 알림

#### 비즈니스 모델
- 💰 방문 요양: 800,000원/월/1인 (주 5회 × 2시간)
- 📊 1,000명 고객 기준: 8억원/월

#### 예상 소요 시간
- **3주**
