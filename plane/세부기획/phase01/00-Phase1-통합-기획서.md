> **Phase 1 목표**: 일상 건강 관리 + 의료 정보 제공

### 1. 복약 관리 시스템 ⭐⭐⭐

#### 개발 범위
```typescript
[약 수동 등록]
  - 이름, 용량, 복용시간
  - 복용 기간 설정
  ↓
[복용 시간 푸시 알림]
  - "오전 9시, 혈압약 드실 시간입니다"
  - 스누즈 (10분 후)
  ↓
[복용 완료 체크]
  - 시니어 앱 체크
  - 가족 앱 자동 알림
  ↓
[약 재고 자동 계산]
  - 초기 재고 - 복용 횟수

  ↓
[복약 캘린더]
```

## 🔑 Phase 1 핵심 전략

### 1. 통합 플랫폼의 가치 경험
```
하나의 앱에서:
  복약 관리 (의)
+ 식단 기록 (식)
+ 안전 점검 (주)
+ 택시 호출 (이동)
+ 가족 모니터링
─────────────────
= "우리 집 의식주 매니저"
```

### 2. 정보 제공으로 빠른 런칭
```
실제 주문 (X) → 업체 정보 제공 (O)
BD 협상 (X) → 공공데이터 활용 (O)
→ 빠른 런칭 → 사용자 피드백 → Phase 2 수요 검증
```

### 3. 가족 안심 = 핵심 가치
```
실시간 알림
+ 통합 대시보드
+ 활동 타임라인
─────────────────
= "부모님 상태 언제든 확인"
= "돌봄 부담 경감"
```

### 4. 시니어 UX 최우선
```
큼직한 UI (60px 버튼)
+ 단순 네비게이션 (최대 2단계)
+ 음성 안내 (TTS)
+ 고대비 색상 (WCAG AAA)
─────────────────
= "시니어도 쉽게 사용"
```
