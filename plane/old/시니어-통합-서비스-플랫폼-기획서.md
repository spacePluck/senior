# 시니어 통합 슈퍼플랫폼 기획서

> **"시니어의 삶 전체를 관리해주는 AI 디지털 집사"**

---

## 📋 프로젝트 개요

### 서비스 정의
시니어의 삶 전체(의식주 + 의료 + 복지 + 안전 + 여가)를 AI와 사람이 협력하여 관리하는 통합 생활 플랫폼

### 핵심 목표
1. 어르신이 복잡한 여러 앱을 사용하지 않고도 필요한 모든 서비스를 한 곳에서 이용
2. 가족과 간병인이 원격으로 부모님/환자를 안심하고 케어할 수 있는 도구 제공
3. 검증된 파트너 업체와 시니어를 신뢰 기반으로 연결
4. AI 기술로 시니어가 스스로 하기 어려운 부분을 보조

### 서비스 철학

**핵심 5원칙:**
1. **어르신은 복잡함을 싫어한다** → 단순한 단일 페이지 기반 설계
2. **스스로 하면 못 하는 부분이 너무 많다** → AI & 사람 기반 하이브리드 서비스
3. **모든 카테고리는 실제 생활의 문제를 해결해야 한다** → 실용성 우선
4. **필요한 서비스가 어디에 흩어져 있든, 여기서 한 번에 해결** → 원스톱 통합
5. **가족·간병인·의료진이 함께 사용하는 플랫폼** → 협업 케어 시스템

## 📊 시장 분석 및 경쟁 환경

### A. 시니어 시장 현황

**시장 규모**
- 국내 60세 이상 인구: 약 1,300만명 (2025년)
- 2030년 예상: 1,800만명 (전체 인구의 35%)
- 시니어 케어 시장 규모: 연 50조원 (2024년 기준)
- 연평균 성장률: 15%

**시장 트렌드**
1. **디지털 시니어 증가**: 60대 스마트폰 보유율 85%, 하지만 활용도는 30% 미만
2. **독거노인 증가**: 전체 시니어의 34% (약 440만명)
3. **장기요양 수요 폭증**: 2025년 100만명 돌파 예상
4. **시니어 소비력 증가**: 월평균 소비 250만원 (자산 보유층)
5. **가족 케어 한계**: 맞벌이 증가로 직접 케어 어려움

**시장 기회**
- 시니어는 있지만 시니어 친화적 서비스는 부족
- 파편화된 서비스들 (병원 예약, 청소, 도시락 모두 다른 앱)
- 가족의 불안감 (부모님 안전/건강 모니터링 수요)
- 정부의 디지털 포용 정책 (예산 지원)

### B. 경쟁사 분석

#### 1. 네이버 (검색 포털)

**서비스:**
- 병원/약국 검색
- 스마트플레이스 (업체 정보)
- 네이버 예약 (일부 서비스)

**강점:**
- 압도적인 사용자 기반
- 풍부한 업체 정보
- 지도 연동

**약점:**
- 시니어 비친화적 UI (작은 글씨, 복잡한 메뉴)
- 광고와 콘텐츠 구분 어려움
- 통합 서비스 아님 (검색만 가능, 예약/결제 별도)
- 케어/모니터링 기능 없음

**차별점:**
→ 우리는 시니어 전용 UI + 통합 서비스 + 가족 모니터링

---

#### 2. 카카오톡 (메신저 기반)

**서비스:**
- 카카오톡 선물하기
- 카카오 헤어샵/뷰티 예약
- 카카오 T (택시)
- 카카오 톡채널 (업체 연결)

**강점:**
- 시니어 포함 전 국민 사용
- 메신저 기반 편의성
- 결제 인프라 (카카오페이)

**약점:**
- 시니어 케어 전문성 없음
- 서비스가 카카오톡 내 파편화
- 의료/건강 기능 부재
- 가족 모니터링 불가

**차별점:**
→ 우리는 시니어 생활 전반을 케어하는 전문 플랫폼

---

#### 3. 당근마켓 (지역 기반)

**서비스:**
- 동네 거래
- 동네 업체 (청소, 수리 등)
- 지역 커뮤니티

**강점:**
- 지역 기반 신뢰
- 동네 업체 연결
- 활성화된 커뮤니티

**약점:**
- 시니어 타겟 아님
- 의료/건강/복지 기능 없음
- 검증되지 않은 개인/업체
- 케어 기능 전무

**차별점:**
→ 우리는 검증된 업체 + 시니어 특화 + 통합 케어

---

#### 4. 시니어 전용 앱들 (실버케어, 시니어톡 등)

**서비스:**
- 건강 기록
- 복약 알림
- 걸음 수 측정
- 간단한 게임

**강점:**
- 시니어 UI (큰 글씨)
- 건강 중심

**약점:**
- 기능 제한적 (건강만)
- 낮은 사용자 기반
- 생활 서비스 연결 없음
- 가족 기능 약함

**차별점:**
→ 우리는 건강 + 생활 + 복지 + 여가 통합 + 강력한 가족 기능

---

#### 5. 요양/복지 서비스 (케어닥, 시니어케어 등)

**서비스:**
- 방문요양 매칭
- 간병인 연결
- 장기요양 신청 대행

**강점:**
- 전문 케어 서비스
- B2B 네트워크

**약점:**
- 요양 등급자만 타겟
- 일상 생활 서비스 없음
- 비싸고 접근성 낮음

**차별점:**
→ 우리는 건강한 시니어부터 요양 필요 시니어까지 전 생애 커버

### C. 경쟁 우위 (Competitive Advantage)

#### 우리만의 강점

**1. 통합성 (All-in-One)**
- 경쟁사: 특정 영역만 (건강 or 쇼핑 or 커뮤니티)
- 우리: 의식주 + 의료 + 복지 + 안전 + 여가 모두

**2. 시니어 특화 UX**
- 경쟁사: 일반 사용자용 UI를 시니어가 억지로 사용
- 우리: 처음부터 시니어 전용 설계 (큰 글씨, 음성, 단순화)

**3. 가족 연동 (핵심 차별점)**
- 경쟁사: 시니어만 또는 가족만
- 우리: 시니어 + 가족이 함께 사용, 실시간 모니터링

**4. AI 기반 예측 케어**
- 경쟁사: 수동 기록
- 우리: AI가 행동 패턴 분석 → 사전 예방

**5. 검증된 파트너 생태계**
- 경쟁사: 불특정 다수 또는 정보만 제공
- 우리: 직접 검증한 신뢰 업체 + 플랫폼 책임

**6. 하이브리드 (AI + 사람)**
- 경쟁사: 완전 자동화 or 완전 수동
- 우리: AI가 보조하고 사람이 실행 (최적의 균형)

---

### D. 시장 진입 전략

**블루오션 영역:**
1. **건강한 시니어 타겟** (대부분 서비스는 요양 필요자만 타겟)
2. **가족 안심 서비스** (시니어 직접 사용 + 가족이 지켜봄)
3. **생활 밀착 서비스** (시계 수리, 냉장고 관리 같은 일상)

**진입 장벽:**
- 높은 수준의 신뢰 필요 → 우리는 검증 시스템으로 해결
- 시니어 교육 필요 → 우리는 오프라인 교육 + 가족 앱으로 해결
- 파트너 확보 어려움 → 우리는 합리적 수수료 + 고객 연결로 해결

## 🎯 타겟 사용자 분석

### 주 사용자: 시니어 (60세 이상)

**인구통계:**
- 연령: 60-90세
- 인구: 약 1,300만명 (2025년 기준)
- 스마트폰 보유율: 85% (하지만 활용도 낮음)

**특성:**
- 앱 다운로드와 회원가입이 어려움
- 작은 글씨를 읽기 힘듦 (노안, 시력 저하)
- 복잡한 메뉴 구조에서 길을 잃음
- 신뢰할 수 있는 업체 찾기 어려움
- 전화 통화를 선호

**Pain Points:**
- "시계 수리점 찾으려면 네이버 검색 → 후기 확인 → 전화번호 복사 → 전화앱 열기 → 붙여넣기... 너무 복잡해"
- "약 먹는 시간 자꾸 잊어버려"
- "냉장고에 뭐가 있는지 모르겠어, 유통기한도 언제인지..."
- "청소 업체 부르고 싶은데 믿을 만한 곳을 모르겠어"
- "자식들한테 매번 물어보기 미안해"

### 부 사용자: 가족 (자녀, 손주)

**특성:**
- 30-50대
- 부모님 건강과 안전이 최우선 관심사
- 직장 생활로 직접 케어 시간 부족
- 원격 모니터링 필요

**Pain Points:**
- "부모님이 약 제대로 드시는지 궁금해"
- "오늘 하루 뭐 하셨는지 알 수 없어"
- "갑자기 쓰러지시면 어떡하지?"
- "병원 예약 대신 해드리고 싶은데 매번 전화하기 번거로워"

### 보조 사용자: 간병인, 요양보호사

**특성:**
- 여러 어르신을 동시에 케어
- 체계적인 관리 도구 필요
- 가족과의 소통 필요

### 페르소나 1: 김영수 (75세, 독거노인)

**기본 정보**
- 성별: 남성
- 거주: 서울 강동구 빌라 1층
- 가족: 아들 1명 (경기 분당 거주), 딸 1명 (미국 거주)
- 건강: 고혈압, 당뇨, 무릎 관절염

12:00 - 도시락 배달 받아서 점심
14:00 - 낮잠
16:00 - 경로당 바둑
18:00 - 저녁 식사 (직접 요리 어려움)
20:00 - TV 뉴스
22:00 - 취침
```

**기술 사용 수준**
- 스마트폰: 갤럭시 A14 (아들이 사줌)
- 사용 앱: 카카오톡 (아들과 연락), 유튜브 (트로트)
- 어려운 것: 앱 설치, 회원가입, 작은 글씨

**Pain Points**
1. **약 관리**: "혈압약, 당뇨약 먹는 시간 자꾸 잊어버려. 아들한테 전화하면 잔소리 들어"
2. **식사 문제**: "혼자 밥 해먹기 귀찮아. 라면만 먹으면 혈압 올라가고..."
3. **외로움**: "하루 종일 말 안 할 때도 있어. 경로당 친구들이라도 없으면..."
4. **병원 예약**: "무릎 아파서 병원 가고 싶은데 예약하려면 전화하기 귀찮아"
5. **안전 불안**: "넘어질까봐 무서워. 아무도 모르면 어떡하지"

**우리 서비스 사용 후**
- 아침 8시 약 알림 → 복용 → 아들 앱에 자동 알림
- 냉장고 사진 찍으면 AI가 "오늘 계란밥 어때요?" 추천
- 경로당 바둑 모임 알림 받고 참여
- 무릎 아프다고 말하면 AI가 정형외과 3곳 추천 → 예약 대행
- 12시간 움직임 없으면 아들에게 긴급 알림

**핵심 니즈**
→ 간편함, 안전, 건강 관리, 외로움 해소

**Pain Points**

5. **품질 불확실**: "청소업체, 수리업체 믿을 만한 곳 찾기 어려워"

**우리 서비스 사용 후**

- 장보기 대행

#### 사용 시나리오

**시나리오 1: 시계 고장**
```

```

**시나리오 2: AI 음성 명령**
```
사용자: "시계 고장났어"
AI: "근처 시계 수리점 3곳을 찾았어요.
     평점이 가장 높은 OO시계방으로 전화 연결할까요?"
사용자: "응"
AI: "전화 연결합니다"
```

- 욕창 위험도 체크

#### 사용 시나리오

- 온도/습도 자동 조절
- 스마트 플러그 (가전 원격 제어)

**모니터링**
- 집 안 움직임 패턴 학습
- 이상 움직임 감지
- 낙상 감지
- 화장실 체류 시간 모니터링
- 12시간 이상 움직임 없으면 긴급 알림

**낙상 예방**
- 집 안 위험 요소 점검 (사진 기반)
- 걸음걸이 분석
- 근력 저하 감지
- 맞춤 운동 처방

**긴급 호출**
- SOS 버튼 (목걸이형, 팔찌형)
- 음성 명령 "도와주세요" → 119 + 가족 동시 연결
- 위치 정보 자동 전송

#### 사용 시나리오

**시나리오 1: 집 안전 점검**
```
1. "주거·안전" → "집 안전 점검"
2. 거실, 화장실, 침실 사진 촬영
3. AI가 위험 요소 분석
   "⚠️ 거실 바닥에 전선 걸려있어요 (낙상 위험)"
   "⚠️ 화장실에 안전바가 없어요"
   "⚠️ 복도가 어두워요"
4. 개선 방법 제안 + 안전 용품 구매 링크
5. 시공 업체 연결
```

**시나리오 2: 긴급 상황**
```
오후 3시 - 어르신이 화장실에서 넘어짐

### 1. AI 매니저 (대화형 AI 집사)

**목표:** 시니어가 말로만 모든 작업을 할 수 있도록

**주요 기능:**

- 복합 작업 자동 처리
- 맥락 이해 및 대화 유지
- 감정 분석 (외로움, 우울 감지)

**대화 예시:**
```
사용자: "시계가 고장났어"
AI: "근처 시계 수리점 3곳을 찾았어요.
     평점이 가장 높은 OO시계방으로 전화 연결할까요?"

사용자: "어제 약 먹었어?"
AI: "네, 어제 오후 2시에 혈압약 드셨어요.
     오늘은 아직 안 드셨는데, 지금 드시겠어요?"

사용자: "내일 뭐 해야 돼?"
AI: "내일은 오후 3시 30분에 OO정형외과 예약 있어요.
     출발 30분 전에 알려드릴까요?"

사용자: "외로워"
AI: "요즘 외출을 안 하셨네요.
     내일 복지관에서 노래 교실이 있는데 가볼까요?
     같이 가실 분도 소개해드릴 수 있어요."
```

**특화 기능:**
- 난청 고려 큰 소리 응답
- 사투리 인식 (지역별)
- 반복 설명 가능
- 천천히 발화

### 2. Vision AI (이미지 인식)

#### A. 냉장고 인식 시스템

**목표:** 냉장고 사진만 찍으면 관리 끝

**기능:**
1. 식재료 자동 인식
   - 계란, 우유, 야채, 반찬 등
   - 수량 파악
   - 브랜드/종류 구분

2. 유통기한 예측
   - 포장지 날짜 OCR 인식
   - 식품별 평균 유통기한 추론

3. 레시피 매칭
   - 현재 재료로 만들 수 있는 메뉴
   - 조리 난이도별 정렬
   - 부족한 재료 1-2가지만 필요한 레시피 우선

4. 장보기 리스트
   - 자주 구매하는 품목 자동 추가
   - 유통기한 임박/소진 예상 품목

**사용 흐름:**
```
1. 냉장고 사진 촬영
   ↓
2. AI 분석 (3초 이내)
   ↓
3. 결과 표시:
   "계란 2개, 우유 1개, 토마토 3개, 김치 반찬"
   "⚠️ 우유 내일 유통기한"
   ↓
4. 추천 메뉴:
   - 계란덮밥 ⭐⭐⭐ (쉬움)
   - 토마토 달걀볶음 ⭐⭐ (보통)
   ↓
5. 장보기 필요:
   "밥, 간장, 참기름 필요해요"
```

#### B. 집 안전 점검 AI

**목표:** 낙상 위험 사전 제거

**기능:**
1. 위험 요소 검출
   - 바닥 케이블/전선
   - 미끄러운 타일
   - 높은 문턱
   - 어두운 조명
   - 계단/욕실 안전바 부재
   - 카펫 끝 말림

2. 위험도 점수
   - 각 공간별 0-100점
   - 전체 집 안전도 등급 (A-F)

3. 개선 방안
   - 우선순위별 정리
   - 필요한 안전 용품 추천
   - 시공 업체 연결

**사용 흐름:**
```
1. 집 안 사진 촬영 (거실, 화장실, 침실)
   ↓
2. AI 분석
   ↓
3. 위험 요소 표시 (빨간 박스)
   "⚠️ 거실: 바닥 전선 (낙상 위험 높음)"
   "⚠️ 화장실: 안전바 없음 (중요)"
   "⚠️ 복도: 조명 어두움 (야간 위험)"
   ↓
4. 안전 점수: 65점 (D등급)
   ↓
5. 개선 방안:
   1순위: 화장실 안전바 설치 (10만원)
   2순위: 전선 정리 (무료)
   3순위: LED 센서등 설치 (3만원)
```

### 3. 행동 패턴 분석 AI

**목표:** 건강 이상 조기 발견

**수집 데이터:**
- 일일 움직임 (모션센서 or 앱 사용 로그)
- 식사 시간/횟수
- 외출 빈도
- 약 복용 시간
- 수면 패턴
- 화장실 방문 횟수

**분석 로직:**
```
평소 패턴 학습 (2주 이상)
    ↓
이상 징후 감지
- 움직임 20%↓ → "활동량 감소, 우울증 의심"
- 식사량 30%↓ → "입맛 없으세요? 소화기내과 예약할까요?"
- 외출 0회 7일 → "외로우실 수 있어요, 복지관 프로그램 추천"
- 화장실 10회↑ → "방광염 의심, 비뇨기과 가볼까요?"
- 야간 화장실 5회↑ → "전립선 검사 권장"
    ↓
가족에게 알림 + 서비스 자동 추천
```

**예측 모델:**
1. **낙상 위험도** (0-100점)
   - 걸음걸이 패턴
   - 근력 추정
   - 집 환경 안전도
   - 과거 낙상 이력

2. **장기요양 필요도**
   - 일상생활 수행능력
   - 인지 기능
   - 건강 상태
   - 등급 예측 (1-5등급)

3. **입원 위험도**
   - 건강 데이터 추이
   - 3개월 내 입원 확률
   - 예방 조치 제안
