## I. 플랫폼의 핵심 철학 및 미션

### 1.1 바야다 홈헬스케어의 핵심 가치

이 플랫폼은 **공감(Compassion), 탁월함(Excellence), 신뢰(Reliability)**라는 바야다 홈헬스케어의 세 가지 핵심 가치를 바탕으로 합니다.

#### 🤝 공감 (Compassion)
- 고객을 항상 존중하고 귀를 기울임
- 공감을 아낌없이 표현
- 고객의 몸과 마음을 살피고 가족의 적응을 도움

#### 🌟 탁월함 (Excellence)
- 최상의 기술과 역량 유지
- 올바른 판단력 배양
- 끊임없는 교육과 훈련

#### 💙 신뢰 (Reliability)
- 전문 간호 인력 기반의 체계적 서비스
- 질 높은 환자 중심 케어
- 투명한 서비스 제공

### 1.2 타겟 고객

**주 대상**: 중증 질환이나 장애 진단으로 생활에 갑작스러운 변화가 생긴 고객
- 뇌졸중, 심혈관 질환 등 중증 질환자
- 장애 진단을 받은 성인 및 노인
- 수술 후 재활이 필요한 환자
- 만성 질환으로 지속적 관리가 필요한 노인
- 독립 생활이 어려운 고령자

**부 대상**: 가족 구성원을 돌볼 때 지지가 필요한 가족
- 원거리 거주 자녀
- 돌봄 부담이 큰 가족
- 전문적 케어가 필요한 환자의 보호자

### 1.3 플랫폼의 미션

> 전 세계 어린이, 성인, 노인이 집에서 독립적으로, 존엄성을 존중받으며 편안하고 안전하게 지낼 수 있도록 돕고, 최고 수준의 홈헬스케어 서비스를 제공

## II. 전문 의료 및 통합 돌봄 모듈

### 2.1 방문 간호 서비스 (케어 메이트)

#### 서비스 개요
**제공 인력**: 정규 간호대학 졸업 + 종합병원 근무 경력이 풍부한 간호사(RN)

**제공 범위**: 의사, 한의사 또는 치과의사가 발급한 **방문간호지시서** 내에서만 제공
- 간호처치
- 재활지원
- 질병예방 교육
- 생활습관 교육

#### 플랫폼 기능 (Phase 1)

##### 2.1.1 방문간호 요청 플로우
```
건강 상태 입력
    ↓
AI 기반 방문간호 필요도 평가
    ↓
방문간호지시서 업로드 (필수)
    ↓
간호사 매칭
    ↓
일정 조율
    ↓
방문 진행
```

#### 요금 체계 (노인장기요양보험 기준)

```typescript
기본 요금 = 노인장기요양보험 수가 기준

부담 비율:
- 공단 부담: 85%
- 수급자 본인 부담: 15%

가산 요금:
- 야간 (22:00-06:00): +30%
- 일요일: +30%
- 유급휴일/근로자의날: +50%
* 중복 가산 없음 (최고 50% 일괄 적용)

예시:
방문간호 1회 (60분) 기본 수가: 50,000원
→ 본인 부담금: 7,500원 (15%)

일요일 방문 시:
50,000원 × 1.3 = 65,000원
→ 본인 부담금: 9,750원 (15%)
```

### 2.2 방문 요양 서비스 (케어 기버)

#### 서비스 개요
**제공 인력**: 노련하고 친절한 요양보호사

**차별성**: 서비스 초기에 **간호사가 함께 방문**하여 체계적이고 전문적인 건강 돌봄 서비스를 설계

**제공 내용**:
- 신체활동 지원 (식사, 배설, 옷 입기 등)
- 일상생활 지원 (청소, 세탁, 장보기 등)
- 인지활동 지원 (말벗, 레크리에이션 등)
- 정서 지원

#### 플랫폼 기능 (Phase 1)

##### 2.2.1 방문요양 신청 플로우
```
장기요양 인정 신청 대행 (무료)
    ↓
등급 판정 대기
    ↓
등급 확정 후 요양보호사 매칭
    ↓
초기 방문 (간호사 + 요양보호사)
    ↓
맞춤 케어플랜 수립
    ↓
정기 방문 시작
```

## VII. 수익 모델

### 7.1 노인장기요양보험 기반 수익

#### 방문 요양
```
1회 방문 (2시간) 기본 수가: 40,000원
- 공단 부담 (85%): 34,000원
- 본인 부담 (15%): 6,000원

바야다 수익 = 공단 부담금 + 본인 부담금
            = 40,000원 (100%)

월 평균 (주 5회 × 4주):
40,000원 × 20회 = 800,000원/월/1인
```

#### 방문 간호
```
1회 방문 (60분) 기본 수가: 50,000원
- 공단 부담 (85%): 42,500원
- 본인 부담 (15%): 7,500원

바야다 수익 = 50,000원

월 평균 (주 3회 × 4주):
50,000원 × 12회 = 600,000원/월/1인
```

#### 통합 서비스 (요양 + 간호)
```
월 수익 = 800,000원 + 600,000원
        = 1,400,000원/1인
```

### 7.2 플랫폼 수수료

#### 디지털 재활 서비스
```
재활 운동 프로그램 (에버엑스)
- 기본 플랜: 무료 (방문 간호 포함)
- 프리미엄 플랜: +50,000원/월
  (1:1 PT 영상 통화, 맞춤 프로그램)
```

#### 원격 의료 수수료
```
원격 진료 (K-DOC) 수수료
- 플랫폼 수수료: 진료비의 10%
```

### 7.5 예상 수익 (1,000명 고객 기준)

```
방문 요양 + 간호 통합:
1,400,000원 × 1,000명 = 14억원/월
```

## VIII. 차별화 포인트

### 8.1 바야다 vs 일반 요양 서비스

| 구분 | 일반 요양 서비스 | 바야다 VIP 플랫폼 |
|-----|----------------|-------------------|
| **초기 방문** | 요양보호사만 | 간호사 + 요양보호사 |
| **케어플랜** | 표준 플랜 | 전문 간호사가 맞춤 설계 |
| **모니터링** | 가족 직접 연락 | 앱으로 실시간 확인 |
| **재활** | 별도 서비스 | AI 재활 운동 포함 |
| **의료 연계** | 없음 | 원격 진료 연동 (Phase 2) |
| **기록 관리** | 종이 | 통합 EMR |

### 8.2 핵심 경쟁력

#### 1. 전문성 (Expertise)
```
간호사 (RN) 중심 케어
= 의학적 판단 + 돌봄 설계
```

#### 2. 디지털 혁신 (Digital Innovation)
```
AI 재활 + 원격 의료 + 통합 EMR
= 디지털 헬스케어 선도
```

#### 3. 통합 돌봄 (Integrated Care)
```
방문 간호 + 방문 요양 + 재활 + 원격 진료
= One-Stop 홈헬스케어
```

#### 4. 글로벌 네트워크 (Global Network)
```
바야다 미국 본사 + 일본 지사
= 글로벌 표준 케어
```

#### 5. 정책 부합 (Policy Alignment)
```
돌봄통합지원법 개정 방향 선제 대응
= 보건의료 + 영양 + 주거 + 일상생활
```
