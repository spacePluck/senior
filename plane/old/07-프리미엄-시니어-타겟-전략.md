> **"자산을 가진 시니어를 위한 프리미엄 생활 관리 플랫폼"**

---

## 📋 개요

### 핵심 인사이트

**현실:**
- 시니어 시장의 **80:20 법칙**
- 상위 20% 시니어가 전체 소비의 70% 차지
- 이들은 **가격보다 품질과 신뢰**를 중시

**우리의 포지셔닝:**
```
저가 플랫폼 ❌
대중적 플랫폼 ❌
→ "프리미엄 시니어 생활 관리 서비스" ✅
```

## 🎯 프리미엄 시니어 정의

### 타겟 프로필

#### Tier 1: 하이엔드 시니어 (상위 5%)
```
연령: 60-75세
자산: 10억 이상 (부동산 포함)
월 가용 소득: 500만원 이상
거주지: 강남, 서초, 송파, 분당, 판교 등

특징:
- 은퇴 전 고위직 (대기업 임원, 의사, 변호사, 사업가)
- 경제적 여유 있음
- 자녀도 경제적으로 성공
- 디지털 리터러시 높음 (스마트폰 능숙)
- "가격보다 가치" 중시

Pain Points:
- "좋은 서비스를 찾기 어려워"
- "믿을 수 있는 업체가 없어"
- "일일이 알아보기 귀찮아"
- "나이 들어도 품위 있게 살고 싶어"
```

#### Tier 2: 준프리미엄 시니어 (상위 15%)
```
연령: 60-80세
자산: 5-10억 (부동산 포함)
월 가용 소득: 300만원
거주지: 중산층 밀집 지역

특징:
- 안정적 연금 수령
- 중산층 생활 유지
- 가끔 고급 서비스 이용
- 자녀 지원 가능

Pain Points:
- "가끔은 좋은 서비스 받고 싶어"
- "특별한 날엔 프리미엄 서비스"
- "건강에는 돈 아끼지 않아"
```

## 💎 프리미엄 서비스 라인업

### 1. 프리미엄 의료 컨시어지

#### 서비스 내용
```
[VIP 건강 관리 패키지]
월 회비: 500,000원

포함 사항:
✅ 전담 건강 매니저 배정 (간호사 자격)
✅ 월 1회 방문 건강 체크
✅ 24시간 긴급 의료 상담 핫라인
✅ 대학병원 VIP 진료 예약 대행
✅ 병원 동행 서비스
✅ 검진 결과 상세 해석 및 상담
✅ 맞춤형 건강 식단 추천
✅ 약 배달 및 복약 관리

부가 서비스:
- 해외 의료 관광 (미국, 일본 암 치료)
- 프리미엄 건강검진 (서울대병원 VIP)
- 성형/미용 시술 예약 (유명 병원)
```

**페르소나 예시:**
```
이철수 (65세, 前 대기업 임원)
자산: 15억 (강남 아파트 소유)
건강 상태: 양호, 예방 건강관리 중요시

"매년 정기 건강검진 받는데, 결과지만 받고
 정확히 뭔지 모르겠어. 전문가가 상세히
 설명해주고 관리해줬으면 좋겠어."

→ VIP 건강 관리 패키지 가입
→ 전담 매니저가 검진 결과 1:1 상담
→ 운동/식단 맞춤 관리
→ 3개월 후 수치 개선, 만족도 ★★★★★
```

### 2. 프리미엄 홈케어 서비스

#### 서비스 내용
```
[집사 서비스 - 월 정기]
월 회비: 1,000,000원

포함 사항:
✅ 전담 생활 매니저 (주 3회 방문)
✅ 집안일 전반 관리
   - 청소 (세밀한 청소, 정리정돈)
   - 세탁 (옷 관리, 다림질)
   - 식사 준비 (영양사 자문 식단)
✅ 장보기 대행 (프리미엄 식재료)
✅ 의류 관리 (계절별 옷장 정리)
✅ 각종 예약 대행 (식당, 공연, 여행)
✅ 우편물/택배 정리
✅ 반려동물 케어 (산책, 미용 예약)

부가 서비스:
- 긴급 상황 즉시 대응
- 집 비울 때 집 관리
- 손님 초대 시 케이터링 준비
```

**페르소나 예시:**
```
박정희 (70세, 前 변호사)
자산: 20억
혼자 거주 (배우자 사별, 자녀 미국)

"집이 넓은데 혼자 관리하기 힘들어.
 믿을 수 있는 사람이 정기적으로
 집안일 해줬으면 좋겠어."

→ 집사 서비스 가입
→ 매주 월/수/금 전담 매니저 방문
→ 청소, 식사 준비, 각종 잡일 처리
→ "마치 호텔 서비스 같아" 만족
```

```

✅ 여유로운 일정 (휴식 시간 충분)
✅ 소규모 프라이빗 (8명 이하)
✅ 의료진 동행 (안전)
✅ 한식 or 맞춤 식사
✅ 휠체어/지팡이 지원
✅ 호텔 엘리베이터/편의시설 사전 체크

```

```
- 우선 예약
- 전담 CS
- 포인트 2배 적립

Premium (월 500,000원)
- VIP 건강 관리
- 24시간 상담
- 프리미엄 매장 우선 이용

VIP (월 1,000,000원+)
- 전담 매니저 (오프라인)
- 컨시어지 서비스 무제한
- 모든 프리미엄 서비스 우선권
```

```
[플래티넘 패키지] - 월 2,500,000원

→ 단독 구매 대비 20% 할인
```

## 🎯 마케팅 전략

### 타겟 접근 방법

#### 1. 오프라인 중심
```
장소:
- 고급 아파트 단지 (강남, 분당 등)
  → 아파트 입주민 대상 설명회
- 프리미엄 골프장/헬스클럽
  → 협찬 이벤트
- 백화점 VIP 라운지
  → 팝업 부스 운영
```

#### 2. 추천 마케팅
```
"한 분이 친구 2명 데려오시면
 다음 달 서비스 비용 50% 할인"

프리미엄 시니어는 네트워크가 강함
→ 입소문 효과 극대화
```

#### 3. 자녀 타겟 광고
```
"부모님께 최고의 선물을
 매달 드리세요"

타겟:
- 40-50대 고소득 자녀
- 강남/판교 직장인
- 외국계 기업

채널:
- 네이버/카카오 타겟 광고
- 페이스북 (40-50대)
- 인스타그램 (스토리 광고)
```
