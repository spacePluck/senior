# Phase 1 최종 개발 우선순위
## VIP 시니어 통합 플랫폼 (6개월)

> **목표**: 바야다 전문 홈헬스케어 (핵심) + 헬스케어 자가 관리 (확장) 통합 개발

---

## 📊 개발 전략

### 핵심 + 확장 통합 개발
```
Phase 1 = 바야다 전문 홈헬스케어 (중증 환자용)
         + 헬스케어 자가 관리 (일반 시니어용)
         + 가족 모니터링 (공통)

→ 하나의 통합 플랫폼에서 모든 시니어 커버
```

### Phase 2로 미루기
```
- 에버엑스 AI 재활 (외부 API 연동)
- K-DOC 원격 진료 (의료법 검토)
- DKSH 물류 (B2B 계약)
- 생활 서비스 실제 주문 (도시락, 청소, 이동 - BD 필요)
```

### 3순위: 복약 관리 시스템 ⭐⭐⭐
**모든 시니어 공통 니즈** (헬스케어 자가 관리)

#### 개발 이유
- 중증 환자 + 일반 시니어 모두 필요
- BD 없이 즉시 개발 가능
- 가족 안심의 핵심 기능

#### 개발 범위
```typescript
[약 수동 등록]

  ↓
[복용 시간 푸시 알림]
  ↓
[복용 완료 체크]
  - 시니어 앱에서 체크
  - 가족 앱 자동 알림
  ↓
[약 재고 자동 계산]
  - 초기 재고 - 복용 횟수
  - 5일 남으면 알림
  ↓
[복약 캘린더]
```

### 8순위: 케어플랜 수립 시스템 ⭐⭐

```typescript
[초기 방문 후]
  ↓
[간호사가 케어플랜 작성] (파트너 앱)

  - 주 진단 선택

  - 단기 목표 (3개월)

  - 방문 횟수 (주 3회/5회/7회)

  - 제공 서비스 체크

  위험도 평가:
  - 낙상 위험: 🟡 중간
  - 욕창 위험: 🟢 낮음
  ↓
[케어플랜 승인] (시니어/가족)
  ↓
[케어플랜 기반 서비스 제공]

[3개월마다 재평가 알림]
```

- 목표 설정
- 서비스 계획
- 케어플랜 조회 (시니어/가족)

- 재평가 알림

## 🔑 핵심 전략

### 1. MVP로 빠르게 시작
```
자동 매칭 (X) → 관리자 수동 매칭 (O)
완벽한 AI (X) → GPT API 활용 (O)
→ 빠른 런칭 → 고객 피드백 → 점진적 개선
```

### 2. 바야다 + 일반 시니어 통합
```
중증 환자: 방문 간호/요양
일반 시니어: 복약 관리, 건강 기록
→ 하나의 플랫폼에서 모든 시니어 커버
```

### 3. 가족 안심 = 핵심 가치
```
실시간 알림 + 통합 대시보드
= 부모님 상태 언제든 확인
= 돌봄 부담 경감
```

### 4. 시니어 UX 최우선
```
큼직한 UI + 단순 네비게이션 + 음성 안내
= 시니어도 쉽게 사용
```
