# 오프라인 매장 등록 및 검증 시스템 기획서

> **"웹사이트 없는 동네 가게도 플랫폼에서 신뢰받는 파트너로"**

---

## 📋 개요

### 배경
- 대부분의 시니어 친화 서비스(청소, 수리, 도시락, 이미용 등)는 **오프라인 소상공인**이 운영
- 웹사이트가 없는 매장: 약 70% 이상
- 기존 플랫폼의 문제점:
  - 네이버 예약: 웹사이트 필수, 온라인 결제 시스템 필요
  - 쿠팡이츠/배민: 음식점 중심, 서비스업 등록 어려움
  - 당근마켓: 검증 없음, 신뢰도 낮음

### 목표
1. **웹사이트 없는 오프라인 매장도 쉽게 등록** 가능
2. **플랫폼의 철저한 사전 검증**으로 시니어에게 신뢰 제공
3. **매장 인증코드 시스템**으로 방문 확인
4. **오프라인 결제 + 온라인 정산** 병행 지원

## 🎯 핵심 시나리오

### Case 1: 동네 이발소 "명성이발관" (웹사이트 없음)
**현황:**
- 운영: 김명성 (65세), 40년 경력
- 위치: 서울 강동구 천호동
- 주고객: 시니어 남성
- 예약 방식: 전화 또는 직접 방문
- 웹사이트: 없음
- 결제: 현금 또는 카드 (직접 결제)

**플랫폼 등록 과정:**
1. **플랫폼 영업팀 방문** → 사업자등록증, 면허증 확인
2. **매장 정보 대신 입력** (사진, 운영시간, 가격표)
3. **시니어 친화 인증** (의자 높이, 손잡이, 편의시설 체크)
4. **매장 고유 인증코드** 발급: `MS-2025-001`
5. **파트너센터 앱 설치** (간단한 사용법 교육)

**서비스 이용 흐름:**
```
시니어 앱에서 "이발" 검색
  → "명성이발관" 선택 (평점 4.8, 시니어 친화 인증)
  → 예약 시간 선택 (11시)
  → 예약 완료 → 매장에 자동 알림

당일 방문:
  → 시니어 앱에서 "도착" 버튼 클릭
  → 매장 인증코드 표시: "MS-2025-001"
  → 매장 사장님이 파트너앱에서 확인 → 서비스 시작

이발 완료 후:
  → 현장 결제 (현금 15,000원)
  → 사장님이 파트너앱에서 "서비스 완료" 클릭
  → 시니어에게 포인트 적립 (500P)
  → 가족 앱에 알림: "아버지 이발 완료 (명성이발관)"
```

## 🏢 매장 등록 프로세스

### 1단계: 사전 심사 (플랫폼 주도)

#### A. 매장 발굴 방법
**플랫폼 영업팀이 직접:**
- 시니어 밀집 지역 (경로당 반경 500m) 매장 방문
- 복지관, 주민센터와 협력하여 추천 매장 리스트 확보
- 시니어 이용자가 직접 추천한 매장 우선 검토

**매장 자체 신청:**
- 플랫폼 웹사이트에서 "파트너 신청" 페이지
- 전화 신청 (대표번호) → 영업팀이 방문 예약

#### B. 필수 서류 확인
**법적 요구사항:**
- [ ] 사업자등록증 (필수)
- [ ] 업종별 면허증 (해당 시)
  - 이발소: 이용사 면허증
  - 음식점: 영업신고증
  - 청소업: 해당 없음 (사업자등록증만)
- [ ] 보험 가입 증명 (배상책임보험 권장)

**매장 정보:**
- [ ] 정확한 주소 및 연락처
- [ ] 운영 시간
- [ ] 가격표 (사진 또는 스캔)
- [ ] 매장 사진 (외관, 내부, 시설)

#### C. 시니어 친화 인증 (체크리스트)
**물리적 환경:**
- [ ] 턱 없음 또는 경사로 설치
- [ ] 화장실 손잡이
- [ ] 의자 높이 (앉고 일어나기 쉬운 높이)
- [ ] 충분한 조명 (어두운 곳 없음)
- [ ] 미끄럽지 않은 바닥

**서비스 태도:**
- [ ] 큰 소리로 또렷하게 말하기
- [ ] 설명 천천히 반복
- [ ] 예약 시간 지키기
- [ ] 과도한 영업 안 함

**등급 부여:**
- ⭐⭐⭐ (3단계): 기본 요건 충족
- ⭐⭐⭐⭐ (4단계): 시니어 친화 환경 우수
- ⭐⭐⭐⭐⭐ (5단계): 시니어 전문 매장

#### D. 계약 조건
**수수료 구조:**
- 서비스 완료 금액의 **8-12%** (업종별 상이)
- 매월 1회 정산 (익월 10일)
- 초기 1개월 수수료 면제 (프로모션)

**플랫폼 제공 사항:**
- 파트너센터 앱 (무료)
- 매장 홍보 (플랫폼 내 노출)
- 결제 인프라 (선택사항)
- 예약 관리 시스템

## 🔐 매장 인증코드 시스템

### 인증코드란?
**목적:**
- 시니어가 실제로 해당 매장에 방문했는지 확인
- 매장과 플랫폼 간 신뢰 보장
- 가짜 예약 및 부정 사용 방지

### 인증코드 구조
**형식:**
```
[매장 코드]-[연도]-[일련번호]
예: MS-2025-001

MS: 명성이발관 약자
2025: 등록 연도
001: 매장 고유 번호
```

**특징:**
- 매장당 1개의 고유 코드
- QR 코드로도 생성 (스캔 가능)
- 매장 입구에 포스터 부착

## 🔍 매장 검증 시스템

### 지속적 품질 관리

#### A. 정기 점검 (분기별)
**플랫폼 담당자 방문:**
- 위생 상태 확인
- 시니어 친화 시설 유지 여부
- 가격 변동 확인
- 사장님 인터뷰 (어려운 점, 개선 요청)

#### B. 시니어 후기 모니터링
**자동 경고 시스템:**
```
평점 4.0 이하 시:
  → 플랫폼 담당자가 매장 연락
  → 문제 파악 및 개선 요청

평점 3.0 이하 시:
  → 신규 예약 일시 중단
  → 긴급 방문 점검
  → 개선 안 되면 계약 해지
```

**후기 분석:**
- 키워드 자동 추출 (예: "불친절", "비쌈", "청결")
- 반복 문제 발생 시 알림

#### C. 미스터리 쇼퍼
**플랫폼이 시니어 모니터 고용:**
- 매월 랜덤으로 매장 방문
- 서비스 품질 체크
- 시니어 친화 태도 평가
- 보고서 작성 → 매장 등급 조정

## 💳 결제 및 정산 시스템

### 결제 방식 (3가지 선택 가능)

#### 1. 현장 결제 (오프라인)
**흐름:**
```
서비스 완료
  → 시니어가 매장에서 직접 결제 (현금 or 카드)
  → 사장님이 파트너센터 앱에서 "현장 결제" 선택
  → 플랫폼 수수료는 월말에 일괄 정산
```

**장점:**
- 매장 입장: 즉시 현금 확보
- 시니어 입장: 익숙한 방식

**플랫폼 수수료 정산:**
- 매월 말일 기준으로 집계
- 익월 10일 매장 계좌에서 자동 이체

#### 2. 온라인 선결제
**흐름:**
```
예약 시 앱에서 결제
  → 플랫폼이 금액 보관
  → 서비스 완료 후 매장에 정산
```

**장점:**
- 시니어: 현금 들고 다닐 필요 없음
- 가족: 대신 결제 가능 (카드 등록)

#### 3. 온라인 후불
**흐름:**
```
서비스 완료
  → 사장님이 "서비스 완료" 클릭
  → 시니어 앱에서 결제 (카드/포인트)
  → 플랫폼이 매장에 정산
```

### 우수 매장 혜택

#### 1. 배지 및 인증
**등급별 배지:**
- 🥉 브론즈: 평점 4.5 이상 + 이용 30건
- 🥈 실버: 평점 4.7 이상 + 이용 100건
- 🥇 골드: 평점 4.9 이상 + 이용 300건

**혜택:**
- 플랫폼 내 상위 노출
- "우수 파트너" 배지 표시
- 수수료 1% 할인

#### 2. 신규 고객 유치 보너스
```
플랫폼을 통한 신규 고객 10명 달성 시:
  → 10,000원 보너스 지급
```

#### 3. 단골 고객 프로그램
```
동일 고객이 3회 이상 재방문 시:
  → 매장에 "단골 고객 보너스" 5,000원
  → 고객에게 할인 쿠폰 자동 발송
```

## 🚨 긴급 상황 대응

### 서비스 중 사고 발생 시

#### 시나리오: 매장에서 시니어가 넘어짐
**1단계: 즉시 조치**
```
사장님이 파트너센터 앱에서:
  → "긴급 상황" 버튼 클릭
  → 자동으로 119 및 플랫폼 콜센터에 연락
  → 가족 앱에 즉시 알림
```

**2단계: 플랫폼 대응**
```
플랫폼 콜센터:
  → 사장님과 통화 (상황 파악)
  → 가족과 통화 (상황 전달)
  → 필요 시 현장 지원팀 파견
```

**3단계: 사후 조치**
```
사고 보고서 작성
  → 원인 분석
  → 매장 책임 여부 판단
  → 보험 처리 (필요 시)
  → 재발 방지 대책 수립
```
