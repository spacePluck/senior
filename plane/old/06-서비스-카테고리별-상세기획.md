# 서비스 카테고리별 상세 기획서

> **"시니어의 의식주부터 의료, 여가까지 모든 생활 서비스"**

---

## 📋 개요

이 문서는 플랫폼에서 제공하는 주요 서비스 카테고리를 상세하게 정의합니다.
각 카테고리별로:
- 제공 서비스 목록
- 오프라인 매장 vs 방문 서비스 구분
- 예약 프로세스
- 특수 고려사항

---

## 🍚 1. 식사 서비스

### 1-1. 도시락/반찬 배달

#### 서비스 유형
**업체 종류:**
- 오프라인 반찬 가게 (배달 지원)
- 도시락 전문 업체
- 시니어 식단 전문 (저염식, 당뇨식)

#### 주요 기능
```
[도시락 주문 흐름]

1. 카테고리 선택: "식사 > 도시락 배달"

2. 식단 선택:
   - 일반 도시락 (7,000원)
   - 저염식 도시락 (8,000원)
   - 당뇨식 도시락 (8,500원)
   - 죽/미음 (6,000원)

3. 배달 시간 선택:
   - 아침 (07:00 - 08:00)
   - 점심 (11:30 - 12:30)
   - 저녁 (17:30 - 18:30)

4. 정기 배달 옵션:
   [ ] 매일 배달 (월-일)
   [ ] 평일만 (월-금)
   [ ] 주 3회 (월/수/금)

5. 결제:
   - 현금 (배달 시)
   - 온라인 선결제
   - 정기 배달 시 월 단위 결제

6. 배달 완료:
   → 배달원이 사진 촬영 (도시락 전달 인증)
   → 시니어 앱 알림
   → 가족 앱 알림: "아버지 점심 도시락 배달 완료"
```

#### 특수 고려사항
```
[알레르기/제외 식재료]
시니어 프로필에 등록:
  - 알레르기: 해산물 ❌
  - 싫어하는 음식: 가지 ❌
  - 선호 음식: 나물 반찬 ✅

→ AI가 자동으로 적합한 도시락 추천
```

#### 매장 검증 기준
- [ ] 식품위생 인증
- [ ] 조리 시설 위생 점검
- [ ] 시니어 식단 이해도 (저염/당뇨식)
- [ ] 배달 시간 준수율

### 1-2. 식당 (오프라인 방문)

#### 서비스 유형
**매장 종류:**
- 경로 식당
- 시니어 친화 식당 (한식, 중식, 일식 등)
- 카페 (노인 친화)

#### 주요 기능
```
[식당 예약 흐름]

1. 검색: "근처 식당"

2. 추천 매장:
   📍 할매 순대국 ⭐⭐⭐⭐⭐
   💰 7,000원
   📏 200m (도보 3분)

   ✅ 1층, 계단 없음
   ✅ 조용한 분위기
   ✅ "간이 안 세요" 후기

   [예약하기] [전화하기]

3. 예약 (선택사항):
   - 날짜/시간 선택
   - 인원 수
   - 요청 사항: "저염식으로 부탁드려요"

4. 방문 → 인증코드 확인 → 식사

5. 결제:
   - 현장 결제 (일반적)
   - 온라인 결제 (가족이 대신 결제 가능)

6. 포인트 적립
```

#### 매장 검증 기준
- [ ] 위생 등급 우수
- [ ] 시니어 친화 시설 (의자 높이, 화장실 손잡이)
- [ ] 저염 메뉴 제공
- [ ] 큰 글씨 메뉴판

## 🏠 2. 생활 서비스

### 2-1. 청소 서비스 (방문)

#### 서비스 유형
**서비스 종류:**
- 일반 가정 청소 (3시간)
- 대청소 (5시간)
- 주방 집중 청소
- 화장실 집중 청소

#### 주요 기능
```
[청소 예약 흐름]

1. 서비스 선택: "생활 > 청소"

2. 청소 범위 선택:
   [ ] 거실
   [ ] 방 (개수: 2개)
   [ ] 주방
   [ ] 화장실
   [ ] 베란다

3. 청소 종류:
   [ ] 일반 청소 (먼지 닦기, 바닥 청소)
   [ ] 대청소 (세밀 청소, 정리정돈)

4. 날짜/시간:
   - 원하는 날짜 선택
   - 시간대: 오전(09:00-12:00) / 오후(14:00-17:00)

5. 정기 청소:
   [ ] 주 1회 (매주 월요일)
   [ ] 월 2회 (첫째/셋째 주)

6. 특이사항:
   "반려견 있어요 (소형견)"
   "무릎이 안 좋아서 직접 청소 어려워요"

7. 결제:
   - 온라인 선결제 (추천)
   - 현장 결제

8. 서비스 당일:
   → 청소 기사 출발 알림 (30분 전)
   → 도착 알림 (실시간 위치)
   → Before 사진 촬영
   → 청소 진행 (2.5시간)
   → After 사진 촬영
   → 시니어 서명 확인
   → 가족 앱에 Before/After 사진 전송
```

#### 청소 기사 검증
- [ ] 경력 증명 (1년 이상)
- [ ] 신원 조회
- [ ] 교육 이수 (시니어 케어 매너)
- [ ] 배상책임보험 가입

### 2-2. 수리 서비스 (방문)

#### 서비스 유형
**수리 종류:**
- 전기/전자 (TV, 냉장고, 에어컨)
- 배관 (수도, 화장실)
- 가구 (의자, 문짝)
- 소형 수리 (시계, 안경, 우산)

#### 주요 기능
```
[수리 예약 흐름]

1. 카테고리 선택: "생활 > 수리"

2. 수리 대상:
   - 전기/전자
   - 배관
   - 가구
   - 기타

3. 문제 설명:
   [사진 업로드] (선택)
   [음성 녹음] (선택)
   [텍스트 입력]

   예: "냉장고에서 소리가 계속 나요"

4. AI 진단:
   "냉장고 압축기 문제일 수 있어요.
    수리 비용은 대략 5-10만원 예상됩니다.
    기사님이 방문해서 정확히 확인할게요."

5. 기사 매칭:
   → 근처 검증된 기사 자동 배정
   → 출발 알림

6. 방문 → 진단 → 견적 안내 → 수리

7. 결제:
   - 수리 완료 후 현장 결제
   - 또는 온라인 결제

8. 수리 내역:
   → 수리 전/후 사진
   → 교체 부품 내역
   → 보증 기간 안내
```

#### 기사 검증
- [ ] 자격증 (전기기사, 배관기능사 등)
- [ ] 경력 3년 이상
- [ ] 신원 조회
- [ ] 배상책임보험 가입

## 🏥 3. 의료/건강 서비스

### 3-1. 병원 예약

#### 서비스 유형
**병원 종류:**
- 내과, 정형외과, 치과, 안과 등
- 종합병원
- 요양병원

#### 주요 기능
```
[병원 예약 흐름]

1. 증상 선택 (AI 도움):
   "어디가 불편하세요?"
   [ ] 무릎이 아파요
   [ ] 소화가 안 돼요
   [ ] 기침이 나요
   [ ] 눈이 침침해요

   시니어: "무릎이 아파요" (음성 또는 클릭)

2. AI 추천:
   "무릎 통증이시군요.
    정형외과를 추천드려요."

3. 병원 추천:
   📍 천호 정형외과 ⭐⭐⭐⭐⭐
   💰 진료비: 15,000원 예상
   📏 500m (도보 7분)

   ✅ 영수님이 2번 방문한 곳
   ✅ "친절해요" 후기 많음
   ✅ 주차 가능

   [예약하기]

4. 예약:
   - 날짜/시간 선택
   - 진료과: 정형외과
   - 증상 메모: "무릎 관절염, 3주 전부터 통증"

5. 예약 확정:
   → 병원에 예약 정보 전달
   → 시니어 앱 알림
   → 가족 앱 알림: "아버지 내일 병원 예약 (천호 정형외과)"

6. 방문 당일:
   → 출발 알림 (1시간 전)
   → 도착 확인
   → 진료 대기
   → 진료 완료

7. 진료 후:
   → 처방전 자동 전송 (약국 연동 가능)
   → 다음 진료 예약 안내
   → 가족 앱에 진료 내역 공유
```

#### 병원 검증
- [ ] 의료기관 인증
- [ ] 시니어 친화 인증 (편의시설)
- [ ] 대기 시간 관리

## 🛡️ 5. 안전/복지 서비스

### 5-1. 장기요양 신청 대행

#### 주요 기능
```
[장기요양 신청]

1. 신청 자격 확인:
   "65세 이상이신가요?" ✅
   "거동이 불편하신가요?" ✅

   → "장기요양 신청이 가능해요"

2. 필요 서류 안내:
   - 신분증
   - 의사 소견서

3. 플랫폼 대행:
   → 서류 접수 대행
   → 등급 판정 일정 안내
   → 판정 결과 알림

   → 요양원 또는 방문요양 연결
```
